# Writing a Postman replica

###### "[Postman is](https://bit.ly/2Um0Mxg) an API client that makes it easy for developers to create, share, test and document APIs."

#### Techstack:
* HTML;
* CSS;
* JS (plain);
* NPM
    * axios
    * bootstrap

#### Commands used:
> `npm init`
>
> `npm i axios`
>
> `npm i --save-dev snowpack`
>
> `npm i bootstrap`
>
> `npm i @popperjs/core`
> 
> `npm start`
